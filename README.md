# Portafolio Profesional - Caso de Estudio

Proyecto de desarrollo web front-end para la presentación de casos de estudio de diseño y maquetación.

## 🚀 Tecnologías Utilizadas

- HTML5 (Semántico)
- CSS3 (Variables nativas, Flexbox, Grid y Media Queries)
- JavaScript (ES6+, DOM, IntersectionObserver)

## 📁 Estructura de Archivos (Cumpliendo Requisitos)

- `index.html` - Página principal del portafolio.
- `proyecto-detalle.html` - Página dinámica para el Caso de Estudio (localStorage).
- `css/main.css` - Estilos globales y específicos unificados.
- `css/variables.css` - Variables de color (Beige, Negro suave, Accent).
- `js/main.js` - Lógica de renderizado dinámico, filtros y animaciones.
- `js/proyectos.js` - Base de datos de los 6 proyectos en formato JS.
- Archivos HTML individuales para los 6 proyectos independientes (`miso-bakery.html`, etc.).

## 🤖 Declaración de Uso de Inteligencia Artificial (IA)

En cumplimiento con las directrices académicas, se declara el uso de asistentes de IA (Gemini) exclusivamente para:

1. Optimización y refactorización del flujo de control en la carga dinámica de datos a través de `localStorage`.
2. Estructuración y modularización de archivos CSS respetando el diseño conceptual previo.
3. Generación de textos ficticios de relleno técnico en idioma español para los casos de estudio.
   _Todo el diseño visual, paleta de colores beige/negro y la lógica arquitectónica original fueron concebidos por la autora del proyecto._
